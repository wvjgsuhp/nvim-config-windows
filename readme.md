# Neovim Config (Windows)

## Installation

1.  Dependencies

  - Telescope

    Install `rg`,

    ```sh
    choco install rg
    ```

  - fzf

    Install `ag`,

    ```sh
    choco install ag
    ```

    Install `fzf`,

    ```sh
    choco install fzf
    ```

2. Install

    ```sh
    pip install -r requirements.txt
    nvim
    ```
